# Electric Field Calculator

A C program that calculates electric field, source charge, and distance using Coulomb's Law.

Features:
- Dynamic memory allocation
- Automatic array resizing with realloc
- SI unit conversions
- Multiple stored calculations
- Valgrind-clean memory management

Concepts Used:
- Structs
- Pointers
- Dynamic arrays
- realloc
- String handling
- Physics calculations
